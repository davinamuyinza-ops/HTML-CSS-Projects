CSS Grid Layout – Technical Documentation

Introduction

CSS Grid Layout is a powerful layout system in CSS that allows web developers to build complex and responsive web layouts with ease. Unlike older layout techniques like floats or inline-block, CSS Grid is designed specifically for laying out elements in rows and columns.

What is a Grid container?

A grid container is an element that has the property display: grid; or display: inline-grid;. Once this is applied, all of its direct children become grid items. The container establishes the grid structure by defining how many columns and rows it contains, as well as how items are spaced and aligned.

Grid Items

Grid items are the direct children of a grid container. They can be automatically placed or explicitly positioned using properties like

  • grid-column-start
  • grid-row-end
  • grid-area
Items can also be aligned individually using
  • justify-self
  • align-self

Grid Lines

Grid lines are the lines between columns and rows in a grid. Items can be positioned by referring to these line numbers. For example

  • grid-column: 1 / 3; means the item starts at line 1 and spans to line 3.

Grid Areas

Named grid areas provide semantic control over layouts. Using grid-template-areas developers can define regions of the grid and place items using grid-area. This approach is especially useful for layouts like headers, footers, and sidebars.

Common Grid Properties

Here are some common properties used in CSS Grid:

  • display: grid Turns an element into a grid container
  • grid-template-columns Defines column structure
  • grid-template-rows Defines row structure
  • gap or grid-gap – Adds space between grid cells
  • justify-items Aligns items horizontally
  • align-items Aligns items vertically
  • grid-area Assigns an item to a named area

Why use CSS Grid?

  • CSS Grid makes layout creation more straightforward and powerful.
  • It eliminates the need for float or flexbox workarounds and is well-suited for both components and full-page layouts.
  • It also works well in combination with Flexbox, which is better for one-dimensional layouts.
